SUSY

Install Susy

  gem install susy

make config.rb file

  require "susy"

  project_type = :stand_alone
  http_path = "/"
  sass_dir = "sass"
  css_dir = "stylesheets"
  images_dir = "img"
  fonts_dir = "fonts"
  javascripts_dir = "js"
  line_comments = false
  preferred_syntax = :scss
  output_style = :expanded
  relative_assets = true

Import susy at the begining of .scss file

  @import "susy";

Make fluid container in .scss

  body {
  @include container(80%);
  }

Make column that will be half size of content

  main{
    @include span(6 of 12);
  }

The same will be

  main{
    @include span(8 of 16);
  }

Making 2 columns

  main{
    @include span(8 of 12);
  }
  asside{
    @include span(4 at 9 of 12);
  }

Maps:key->value pair

  $susy: (
      columns: 12,
      gutters: 1/4,
      math: fluid,
      output: float,
      gutter-position: inside,
  );

Layout

  $widelayout: (
      columns: 24,
      gutters: 1/4,
      math: fluid,
      output: float,
      gutter-position: inside,
  );

  @include layout($widelayout);

Layout shorthand

  @include layout(24 1/4 fluid float inside);
  );
  or
  main{
    @include span(8 at 0 of 24 1/8);
  }
  asside{
    @include span(4 at 9);
  }

Breakpoint install

  gem install breakpoint

In config.rb add requirements

  require "susy"
  require "breakpoint"

Import susy at the begining of .scss file

  @import "susy";
  @import "breakpoint";

Use breakpoint, mobile first

  @include breakpoint(500px){
    main{
      @include span(8 of 12);
    }
    asside{
      @include span(4 at 9 of 12);
    }
  }

Use 3 columns inside main column

  @include breakpoint(1200px){
    @include layout(24);
    main{
      @include span(21);
      .col1, .col2, .col3 {
        @include span(4 of 12)
      }
    }
    asside{
      @include span(3);
    }
  }

Use variables and mixins

  $small:500px;
  @include susy-breakpoint($small,24){
    main{
      @include span(21);
      .col1, .col2, .col3 {
        @include span(4 of 12)
      }
    }
    asside{
      @include span(3);
    }
  }
