BOOTSTRAP tips

To start use Boostrap must add CDN link in header and .js library at the booton of the body tag.

Bootstrap defaults

font-size is 14px and line-height is 1.428 applied to body and paragraphs

Emphasis

Class lead will give larger font size, lighter weight and a taller line height.

<p class="lead"> ... </p>

Small tag set text at 85% the size of the parent

<small>some text</small>

Strong tag is rendered as bold text

<strong>some text</strong>

Em tag is rendered as italic text

<em>some text</em>

U tag is rendered as underline text

<u>some text</u>

Alignment class

Align text to left

class="text-left"

Align text to center

class="text-center"

Align text to right

class="text-right"

Transformation Classes

Lowercased text

<p class="text-lowercase">Some Text</p>

Uppercased text

<p class="text-uppercase">Some Text</p>

Capitalized text, first letter of the word

<p class="text-capitalize">Some Text</p>

Media Queries

Extra small devices (xs - less than 768px)

No media query, since this is the default in Bootstrap

Small devices (sm - 768px and up)

@media (min-width: @screen-sm-min)
{...}

Medium devices (md - 992px and up)

@media (min-width: @screen-md-min)
{...}

Large devices (lg -1200px and up)

@media (min-width: @screen-lg-min)
{...}

Class .visible-xs-*

Visible at xs but hidden at sm, md and lg

Class .hidden-xs-*

Hidden at xs but visible at sm, md and lg

Floats

Float left or rgiht

<div class="pull-left">...</div>
<div class="pull-right">...</div>

Center Content Blocks

Set an element to display:block and center via margin

<div class="center-block">...</div>

Clearifx

<div class="clearfix">...</div>

Showing and Hiding Content

<div class="show">...</div>
<div class="hidden">...</div>

Image Replacement

Replace an element's text contetn with a backgound image

<h1 class="text-hide">Heading<h1>

Screen Reader

Hide an element to all devices except screen readers

<a class="sr-only" href="#">some text</a>

FORMS

All textual input, texarea and select with class .form-control have 100% width.

Bootstrap provides 3 types od form layouts: Vertical form(default), Horizontal form, Inline form

Standard Form roules

<form role="form">
  <div class="form-group">
    <input class="form-control" ...>

Supported Form Controls

Bootstrap supports the following form controls: input, textarea, checkbox, radio and select

Most common form control types are:text, password, datetime, datetime-local, date, month, time, week, number, email, url, search, tel and color.

Inputs will be only fully styled if their type is properly declared.

<input type"text" class="form-control" placeholder="Text Input">
